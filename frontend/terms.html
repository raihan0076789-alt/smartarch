Section 02

Service Description

SmartArch is an AI-powered architectural design and project management web application. The platform provides the following core services:

  • AI Floor Plan Generation: Automatic generation of architectural floor plans using the Ollama phi3:mini AI model and binary space partitioning algorithms based on user-submitted design parameters.
  • Floor Plan Image Analysis: Upload and analysis of existing floor plan images using edge detection to extract and digitize room layouts.
  • 2D and 3D Visualization: Interactive rendering of architectural designs as two-dimensional blueprints and three-dimensional building models, with export capabilities in OBJ and GLTF formats.
  • Project Management: Creation, storage, organization, and version tracking of architectural design projects with status management and collaboration tools.
  • Cost & Build Estimation: Automated calculation of total floor area, estimated construction costs, and projected build timelines based on project dimensions.
  • Admin Dashboard: A separate administrative interface for authorized administrators to manage users, monitor platform activity, and view analytics.

We reserve the right to modify, suspend, or discontinue any feature or service component at any time with reasonable notice where practicable.
