Platform Capabilities

Everything You Need to
Build Great Architecture

A complete architectural design platform — from AI-generated floor plans to project management, 3D visualization, and admin analytics.

AI Engine
AI Floor Plan Generation

Powered by Ollama (phi3:mini), our AI generates intelligent room layouts optimized for your space dimensions, building type, and floor count using a BSP layout algorithm.

Visualization
2D & 3D Model Rendering

Instantly switch between precise 2D floor plan views and interactive 3D building models. Export designs in JSON, OBJ, and GLTF formats for use in other tools.

Project Hub
Full Project Management

Create, save, and organize multiple architectural projects. Track status (draft, in progress, review, approved), add collaborators, and maintain full version history.

Security
JWT Authentication & Roles

Secure user registration and login with bcrypt password hashing and JWT tokens. Three role levels — user, architect, and admin — with route-level access control.

Administration
Admin Dashboard & Analytics

A full-featured admin panel with real-time stats, user/project management, growth charts, activity feeds, and top contributors — all protected by admin-only middleware.

Intelligence
Smart Auto Layout Engine

Specify rooms by type and count; the BSP-based algorithm automatically arranges them for optimal space utilization, handling proportions, adjacency, and structural flow.

Input
Floorplan Image Analysis

Upload an existing PNG/JPEG floor plan image; edge detection algorithms automatically extract room outlines and generate a structured digital floor plan from your sketch.

Estimation
Cost & Build Estimation

Automatically calculate total floor area, room count, estimated construction cost, and projected build time based on your design's dimensions and floor configuration.

System Workflow

From Idea to
3D Reality — in Minutes

01

Register & Set Project Parameters

Create your account and define your project — name, building type (residential, commercial, industrial), dimensions, number of floors, and desired rooms.

02

AI Generates Your Floor Plan

Submit your parameters to the AI backend. The phi3:mini model and BSP algorithm intelligently partition and arrange rooms, generating a complete, optimized layout.

03

Visualize in 2D & 3D

Your floor plan renders instantly as a detailed 2D blueprint and a 3D walkthrough model. Adjust materials, specifications, and room positions interactively.

04

Save, Collaborate & Export

Projects auto-save to MongoDB. Invite collaborators, track revision history, and export finalized designs in OBJ or GLTF for use in professional CAD tools.
