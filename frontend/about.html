About the Project

Building the Future of
Architectural Design

SmartArch is a full-stack, AI-powered platform designed to transform the way architectural planning and residential design are created and managed. By combining intelligent design generation, interactive visualization tools, and professional project management features, the platform provides a seamless environment where users can move from concept to structured architectural planning with greater efficiency.

The system leverages modern technologies to assist users in generating architectural layouts based on specific requirements such as room types, dimensions, and structural preferences. Through its intelligent processing capabilities, the platform analyzes user inputs and produces structured design outputs that help visualize residential layouts and planning ideas in a more organized and accessible way.

The platform is built using a modern full-stack architecture that integrates a responsive frontend interface, a scalable backend system, and a secure database layer. This ensures that the system remains flexible, efficient, and capable of supporting future enhancements such as advanced AI design assistance, expanded visualization capabilities, and collaborative project environments.

Project Overview

What is SmartArch?

SmartArch is a modern web application designed to democratize architectural design. Using artificial intelligence, it enables anyone — from professional architects to first-time homeowners — to generate precise, optimized floor plans and visualize them in stunning 3D.

The system combines a robust Node.js/Express API backend with a locally-running AI model (Ollama phi3:mini) to provide intelligent layout generation, real-time 3D visualization, full project lifecycle management, and a comprehensive administrative control panel — all backed by MongoDB for persistent storage.

AI-Powered Design

Automated floor plan generation using large language models and binary space partitioning algorithms, eliminating manual layout work.

3D Visualization

Real-time interactive 3D model generation from 2D floor plans with material customization and export capabilities.

Team Collaboration

Multi-user project system with role-based access control, collaborator invites, and version history tracking.

Enterprise Security

JWT token authentication, bcrypt password hashing, and middleware-enforced admin routes protect all data and operations.

System Workflow

From Input to Output

The user journey is designed to be fast and intuitive — from account creation to a fully rendered 3D building model in minutes.

01
User Registration & Authentication
Users register with name, email, and password. Bcrypt hashes the password before storage. A JWT token is issued on successful login, stored client-side, and sent with every subsequent API request.
02
Project Configuration
Users create a new project specifying: name, description, building type, total dimensions (width × depth), number of floors, desired rooms by type, materials (exterior, interior), and optional garage/pool specifications.
03
AI Generation (Port 3001)
The frontend sends parameters to the AI backend service. The phi3:mini model processes the request, while the BSP layout engine partitions the floor space and positions rooms based on type, adjacency rules, and area constraints.
04
Visualization & Editing
The generated floor plan renders in 2D (SVG-based blueprint view) and 3D (interactive WebGL model). Users can adjust room positions, change materials, and see cost estimates update in real-time.
05
Save & Persistence
Projects auto-save to MongoDB via the main API. Pre-save hooks recalculate metadata (total area, rooms, cost, build time). Version history is maintained for rollback. Projects can be shared publicly or kept private.
06
Export & Collaboration
Finalized designs export as JSON (structured data), OBJ (3D mesh), or GLTF (for game engines and visualization tools). Collaborators can be invited with viewer or editor roles to contribute to the same project.
